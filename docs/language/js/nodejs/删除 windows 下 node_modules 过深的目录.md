# 删除 windows 下 node_modules 过深的目录

---

## 删除方法

---

- 方法一：

```shell

npm uninstall

```

- 方法二：

如果没有保存到 package.json 的包只能手动添加包名来删除了。

```shell

npm uninstall 包名

```


- 方法三：

多个包都没保存到 package.json 的情况，就这样删除

```shell

for /f %m in ('dir /b node_modules') do npm uninstall %m

```

- 方法四

全部删除

```shell

rm -rf node_modules

```
